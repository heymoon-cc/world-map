# Векторная карта OSM

## Технологии
* Symfony
* Valkey
* GEOS

## Подготовка данных
Карта может быть экспортирована с помощью [extract.bbbike.org](https://extract.bbbike.org) в формате Osmium GeoJSON.

## Команды
* `app:load`: инициализация карты из файла Osmium GeoJSON

## Методы
* `/tiles/{x}/{y}/{z}`: получение карты

## Переменные окружения
| Название | Значение по-умолчанию | Описание                                                                                                                                   |
| ----------- |-----------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| `MID_ZOOM` | `8` | Масштаб до которого будет "прогрет" кеш подготовленных тайлов. При запросе карты с бóльшим масштабом тайлы будут герерироваться по запросу |
| `ALLOW_ORIGIN` | `*` | HTTP-заголовок `Access-Control-Allow-Origin`                                                                                               |
